# Analisis Sentimen dengan Python & TextBlob



Proyek ini melakukan analisis sentimen pada teks menggunakan Python dan library TextBlob dengan mengklasifikasikan teks menjadi Positif, Negatif dan Netral kemudian menampilkan hasilnya dalam bentuk pie chart.

## Fitur

Analisis sentimen teks menggunakan TextBlob.
Menghitung jumlah teks positif, negatif dan netral.
Menampilkan persentase sentimen.
Visualisasi pie chart dari teks yang dianalisis.

## Instalasi


Clone repository ini:
```
git clone https://github.com/username/nama-repo.git
cd nama-repo
```

Instal library yang dibutuhkan:
```
pip install textblob matplotlib
```

Download corpus TextBlob:
```
python -m textblob.download_corpora
```
